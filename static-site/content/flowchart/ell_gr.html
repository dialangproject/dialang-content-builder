Καλωσορίσατε στο DIALANG

Καλωσορίσατε στο DIALANG, το ανοιχτής γραμμής σύστημα διάγνωσης γλωσσικών ικανοτήτων. Το DIALANG θα σας βοηθήσει να ανακαλύψετε πόσο καλοί είστε στην κατανόηση γραπτού λόγου, στην κατανόηση προφορικού λόγου, στην παραγωγή γραπτού λόγου, στο λεξιλόγιο, στη γραμματική, καθώς και με ποιον τρόπο είναι δυνατό να βελτιωθείτε. Ακόμη θα σας βοηθήσει να αξιολογήσετε και να κατανοήσετε τις γλωσσικές ικανότητες που διαθέτετε.

Η ΔΙΑΔΙΚΑΣΙΑ

Το DIALANG αποτελείται από διάφορα στάδια. Πιέστε την παρακάτω εικόνα για να ανακαλύψετε σε τι αναφέρεται κάθε στάδιο. Πατήστε το πλήκτρο της Προώθησης, για να περάσετε στο πρώτο στάδιο.

Το πρώτο πράγμα που πρέπει να κάνετε είναι να επιλέξετε ποια εξέταση τεστ θέλετε να κάνετε. Επιλέξτε οποιονδήποτε συνδυασμό γλώσσας και δεξιότητας από αυτά που είναι διαθέσιμα.
Αυτό το προαιρετικό στάδιο σας βοηθά να προσδιορίσετε κατά προσέγγιση πόσο καλά γνωρίζετε τη γλώσσα. Καλείστε να αποφασίσετε αν οι λέξεις που βλέπετε στην οθόνη είναι υπαρκτές ή έχουν εφευρεθεί για τους σκοπούς της άσκησης. Δεν είναι τόσο εύκολο όσο φαίνεται!
Αυτό το στάδιο είναι προαιρετικό, αλλά αποτελεί μέρος των στόχων που επιδιώκει το DIALANG. Καλείστε να ανταποκριθείτε σε προτάσεις που αφορούν την ικανότητά σας να χρησιμοποιείτε ή να κατανοείτε μια γλώσσα. Αυτή η άσκηση σας παροτρύνει να σκεφτείτε τις δικές σας γλωσσικές ικανότητες, γεγονός που στην πραγματικότητα σας βοηθάει στην αποτελεσματική μάθηση. Μπορεί να έχετε υποτιμήσει ή υπερεκτιμήσει τις ικανότητές σας. Αν έχει συμβεί αυτό, θα σας το πει το DIALANG.
Σ’ αυτό το στάδιο εξετάζεστε στη γλώσσα και στη δεξιότητα που επιλέξατε στην αρχή. Σας επισημαίνουμε ότι αν δεν περάσετε από τα στάδια της κατατακτήριας εξέτασης και της αξιολόγησης, πολύ πιθανό να σας δοθεί ένα τεστ που μπορεί να είναι είτε πολύ δύσκολο είτε πολύ εύκολο για σας. Υπάρχουν διάφοροι τύποι ερωτήσεων: σε μερικές είναι απαραίτητο να πληκτρολογήσετε την απάντησή σας και σε άλλες απλά να επιλέξετε τη σωστή απάντηση από μια σειρά επιλογών. Πρέπει να ολοκληρώσετε το τεστ, γιατί διαφορετικά δε θα σας δοθεί αποτέλεσμα.
Αυτό το μέρος του προγράμματος σας ανατροφοδοτεί με πληροφορίες που βασίζονται στις απαντήσεις που έχετε δώσει. Εκτός από τα αποτελέσματα που προκύπτουν από τη γλωσσική και την κατατακτήρια εξέταση, σας δίνονται χρήσιμες πληροφορίες και συμβουλές ειδικών. Έχετε τη δυνατότητα να ελέγξετε την ορθότητα των απαντήσεών σας. Μπορείτε να καταλάβετε τι ακριβώς σημαίνουν τα αποτελέσματα και να πάρετε συμβουλές για τον τρόπο με τον οποίο μπορείτε να αναπτύξετε περισσότερο τις γλωσσικές σας ικανότητες. Επίσης μπορείτε να βρείτε περισσότερες πληροφορίες για την αυτοαξιολόγησή σας.

Από αυτό το σημείο μπορείτε να εγκαταλείψετε το πρόγραμμα ή μπορείτε να επιλέξετε ένα άλλο τεστ. Αν επιλέξετε ένα διαφορετικό τεστ στην ίδια γλώσσα, δε χρειάζεται να επαναλάβετε την κατατακτήρια εξέταση.
