Μπορώ να γράψω απλές μεμονωμένες φράσεις και προτάσεις.
Μπορώ να γράψω μια απλή κάρτα.
Μπορώ να δώσω μικρές, βασικές περιγραφές γεγονότων και δραστηριοτήτων.
Μπορώ να περιγράψω την οικογένειά μου, τις συνθήκες διαβίωσής μου, το σχολείο μου, την τωρινή ή την πιο πρόσφατη δουλειά μου.
Μπορώ να περιγράψω δραστηριότητες του παρελθόντος και προσωπικές εμπειρίες.
Μπορώ να γράψω πολύ σύντομες εκθέσεις, που δίνουν πληροφορίες για καθημερινά γεγονότα και να δηλώσω τους λόγους των πράξεων.
Μπορώ να περιγράψω βασικές λεπτομέρειες απρόβλεπτων συμβάντων, π.χ. ενός ατυχήματος.
Μπορώ να περιγράψω όνειρα, ελπίδες και φιλοδοξίες.
Μπορώ να πάρω μηνύματα που περιγράφουν πληροφορίες, προβλήματα κτλ.
Μπορώ με συντομία να αιτιολογήσω και να εξηγήσω γνώμες, σχέδια και ενέργειες.
Μπορώ να εξηγήσω μια άποψη πάνω σ’ένα τοπικό ζήτημα δίνοντας τα πλεονεκτήματα και τα μειονεκτήματα των διάφορων γνωμών.
Μπορώ να αναπτύξω ένα επιχείρημα και να δώσω λόγους υπέρ ή κατά μιας συγκεκριμένης άποψης.
Μπορώ να αξιολογήσω διάφορες ιδέες και λύσεις ενός προβλήματος.
Μπορώ να κάνω υποθέσεις για αιτίες, συνέπειες και υποθετικές καταστάσεις.
Μπορώ να επεκταθώ και να υποστηρίξω απόψεις σε κάποια έκταση, εκφράζοντας δευτερεύοντα σημεία, αιτίες και σχετικά παραδείγματα.
Μπορώ να δώσω σαφείς, λεπτομερείς περιγραφές πολύπλοκων θεμάτων.
Μπορώ να δώσω την κατάλληλη και αποτελεσματική λογική δομή, η οποία βοηθάει τον αναγνώστη να βρει τα σημαντικά σημεία.
Μπορώ να γράψω πάντα χωρίς να συμβουλευτώ λεξικό.
