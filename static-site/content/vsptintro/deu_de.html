Einstufungstest

Dieser Test soll den Umfang Ihres Wortschatzes in der Fremdsprache ermitteln. Mit diesem einleitenden Test wird der Schwierigkeitsgrad der Testaufgaben festgelegt, die Ihnen anschließend vorgelegt werden.

Dieser Test legt Ihnen eine Auswahl von \'Wörtern\' vor, die zum Teil tatsächlich existieren, zum Teil aber frei erfunden sind.

Bei diesen \'Wörtern\' handelt es sich um Verben, z.B. \'sprechen\', \'laufen\', \'essen\', usw.

Wenn Sie der Meinung sind, dass das betreffende Wort tatsächlich existiert, so klicken Sie auf das Feld \'Ja\'. Wenn Sie der Meinung sind, dass es sich um ein erfundenes Wort handelt, so klicken Sie auf das Feld \'Nein\'.

Sie müssen diesen Test nicht machen. Sie können den Test auch abbrechen. Sie laufen dann aber Gefahr, Testaufgaben zu bekommen, die zu schwer oder zu leicht für Sie sind. Wir raten Ihnen daher dringend, diesen Test nicht auszulassen.

Warnung: Wenn Sie den Einschätzungstest überspringen, könnten Sie einen Test erhalten, der zu schwer oder zu leicht für Sie ist.

Sind Sie sicher, dass Sie den Einschätzungstest überspringen möchten?
